## Opcao Pelo Site Do GitHub

1. Acesse `https://github.com/new`.
2. Em `Repository name`, coloque `controle-financeiro-pessoal`.
3. Marque como `Public` ou `Private`.
4. Clique em `Create repository`.
5. Na tela do repositorio, envie os arquivos do projeto usando `uploading an existing file`.
6. Use o arquivo ZIP gerado em `outputs/controle-financeiro-github.zip` para copiar os arquivos para uma pasta e enviar.

## Opcao Pelo VS Code

1. Abra esta pasta no VS Code.
2. Instale o Git se o VS Code pedir.
3. Clique no icone de controle de codigo-fonte.
4. Clique em `Publish to GitHub`.
5. Escolha o nome `controle-financeiro-pessoal`.
6. Confirme a publicacao.
