# Fundamentos de Engenharia de Software

Projeto: Controle Financeiro Pessoal

## Estrutura

```text
FUNDAMENTOS_DE_ENGENHARIA_DE_SOFTWARE
├── FRONT_END
│   └── src
├── BACK_END
│   └── src
└── CONTROLE_FINANCEIRO_PESSOAL
    ├── IMAGENS
    └── DOC
```

## Conteúdo

- `FRONT_END/src`: telas, componentes e estilos da aplicação.
- `BACK_END/src`: APIs, Prisma, autenticação, banco de dados e scripts.
- `CONTROLE_FINANCEIRO_PESSOAL/IMAGENS`: imagens das telas do sistema.
- `CONTROLE_FINANCEIRO_PESSOAL/DOC`: PDF do projeto e diagrama simples.

## Tecnologias

- Next.js
- TypeScript
- Prisma
- SQLite
- Node.js
